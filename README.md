### 仿网易云音乐
> 此项目为个人TypeScript学习项目，仿网易云音乐移动端web，实现了网易云音乐移动端的核心功能
#### 技术栈
```
Vue + Vuex + TypeScript + Vant
```

## 安装运行（Build Setup）

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run serve

# build for production with minification
npm run build
```
